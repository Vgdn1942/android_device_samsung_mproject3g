# android_device_samsung_mproject3g
Работает:

- Сеть
- Передача данных
- Wi-Fi
- Bluetoth
- Nfc
- GPS
- Графика
- Подсветка кнопок
- Вибрация кнопок
- Датчики
- Проигрывание аудио
- Проигрывание видео
- Микрофон
- Монтирование карт памяти
- Режимы adb/mass_storage
- USB-OTG

- Камера

- Зарядка выключеного телефона
